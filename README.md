# 3 en Raya - Flask Web App 🎮

Este es un proyecto básico desarrollado en **Python** utilizando el framework **Flask**. El objetivo es crear una aplicación web sencilla para practicar la lógica de juegos, el manejo de sesiones y posteriormente su contenedorización con **Docker**.

## 📋 Características
* Juego clásico de **3 en raya** (Tic-Tac-Toe).
* Lógica de turnos automática (X y O).
* Verificación de ganador y empate en tiempo real.
* Opción de reinicio de tablero.
* Interfaz web limpia usando plantillas **Jinja2**.

## 🛠️ Requisitos
* Python 3.10 o superior.
* Flask.
